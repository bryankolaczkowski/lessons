What is a jupyter notebook?

A jupyter notebook combines instructions and text with editable, executable Python code.

In this course, we will be using jupyter notebooks to manage our programming exercises. Anytime we need to write, edit or execute code, we will do so through a jupyter notebook. Skeleton notebooks will be provided for every course exercise, so you won't need to write your own notebook from scratch. You are free to modify your notebook by adding text notes or other informational resources, and you will often be required to write, edit and/or execute code directly from your notebook.

Jupyter notebooks are not just for this course; they are a very popular mechanism for sharing annotated Python code amongst professionals in a wide variety of fields related to computer programming and data science. Once you learn to use jupyter notebooks effectively, you can use them to access existing content and share your code and data analyses with others.

Although jupyter notebooks are closely associated with the Python programming language and widely used in the Python programming community, jupyter notebooks can also be used to write, edit, execute and share code written in other programming languages. So don't think they are just a 'Python thing'.


What is google colaboratory?

Google's colab is a platform for running jupyter notebooks.

Google colaboratory or 'colab' is an online platform allowing anyone to access, modify, execute and share jupyter notebooks. Google colab connects your jupyter notebooks to high-performance computing resources using only a basic web browser, making it an incredibly powerful tool that can open the door to high-performance computing for users who don't necessarily want to become professional programmers.

If you dig a bit deeper, you'll find that google colab is basically just a specific instance of a platform called "jupyter hub", which is just a general server for sharing jupyter notebooks. So, google colab is a jupyter hub server, and there are other jupyter hub servers that are not google colab. Some of the nice things about colab is that it is free for public use, provides access to high-performance hardware and integrates with github and other repositories that support easy file sharing.

My first jupyter notebook in colab

To access this course's first jupyter notebook, you'll need to log into google colab. Open a web browser and navigate to https://colab.research.google.com.

The first important thing you'll need to do in google colab is to make sure you are logged in using a google account. You can use any google account to complete this course. If you have a @gmail.com email address, you probably already have a personal google account, and you can use that account for this course, if you'd like. If you have a @ufl.edu email address, you probably have UF gatorlink credentials, which include a UF-associated google account; you can use your UF-associated google account to complete this course, if you'd like. If you don't have an existing google account or UF gatorlink credentials, you will probably need to create a new personal google account to complete this course.

Look in the upper right-hand corner of the google colab landing page. If you are already logged into a google account, you should see your account icon in the upper-right corner. If this is the case, you are ready to access the course jupyter notebooks.

If you see a button labelled "Sign in" in the upper-right corner of the google colab landing page, you will need to log into a valid google account in order to access colab and run the course jupyter notebooks.

To log into colab, click on the "Sign in" button; you'll be taken to the google account sign-in page. If you have a google account (or many), you should see your google accounts listed, and you can choose whichever account you'd like to use to complete this course. If you don't have an existing google account, you can create a new account by following the instructions.

If you have a University of Florida email address and gatorlink credentials, you can use your UF credentials to access your existing UF-associated google account. If your UF email address is listed as one of your google accounts, you can click on this account to sign in; you will be taken to the UF login page, where you will need to use your gatorlink id and password to log in, and you will need to go through two-factor authentication.

If you don't see your UF email address listed in your google accounts, and you would like to use your UF-associated google account for this course, click on "Use another account".

Enter your UF email address in the "Sign in" form, and click the "Next" button to authenticate. You will be taken to the UF login page, where you will need to use your gatorlink id, password and two-factor authentication to login.

Once you log in, you should see your account icon in the upper-right corner of the google colab landing page, and you should also see a 'popup' allowing you to access jupyter notebooks through various means.

Now that you're logged into google colab, you need to access the course jupyter notebooks.

To access this course's first jupyter notebook, click on the "GitHub" tab on the popup.

and will not necessarily be the first notebook on the list! Click on the "helloworld.ipynb" name to open this notebook in colab. The popup should disappear, and you should see a page with "welcome to jupyter notebooks!" displayed at the top.
